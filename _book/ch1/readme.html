第一章 BOM

ECMAScript 将 BOM 描述为 JS 的核心，但是实际上 BOM 是使用 JS 开发 Web 应用的核心。

实际上 BOM 是没有一个规范的，只是靠着浏览器开发商的意愿发展起来的，实际上浏览器之间共同的部分就是事实标准。

事实上，如果一个创意点非常好的话其他的厂商会借鉴的，毕竟谁都不想让自己的浏览器更落后更加原始。
